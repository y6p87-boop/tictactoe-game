# Tic-Tac-Toe Game

A two-player Tic-Tac-Toe game built with Python and tkinter.

## Features
- Graphical user interface using tkinter
- Score tracking for both players and draws
- Restart game and reset scores buttons
- Win celebration message display

## How to Run
python game.py

## Requirements
- Python 3.x (tkinter is included by default)

## How to Play
- Two players take turns clicking the grid squares
- Player X always goes first
- First player to get 3 in a row wins
- Use Restart Game to start a new round
- Use Reset Scores to clear all scores
